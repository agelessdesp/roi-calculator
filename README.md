# ROI-калькулятор imot.io

Статический ROI-калькулятор для оценки экономического эффекта от внедрения речевой аналитики imot.io.

## Что лежит в репозитории

- `index.html` — основная публичная точка входа для GitHub Pages.
- `ROI-калькулятор imotio (версия от 14.04).html` — исходный HTML-файл проекта.
- `AGENTS.md` — основная внутренняя документация по структуре и логике проекта.
- `CLAUDE.md` — короткий указатель на `AGENTS.md`.

## Локальный запуск

Откройте `index.html` в браузере.

Так как проект статический и использует CDN-зависимости, отдельная сборка не требуется.

## Публикация через GitHub Pages

1. Откройте репозиторий на GitHub.
2. Перейдите в `Settings -> Pages`.
3. В блоке `Build and deployment` выберите `Deploy from a branch`.
4. Укажите ветку `main` и папку `/(root)`.
5. Сохраните настройки и дождитесь публикации.

После этого сайт будет доступен по адресу формата:

`https://agelessdesp.github.io/roi-calculator/`

## Замечания

- Проект использует кириллицу в именах файлов, поэтому для публикации и публичной ссылки используется `index.html`.
- Основная логика находится в одном HTML-файле с inline CSS и inline JavaScript.
- При правках важно проверять не только интерфейс, но и расчёты, сценарии и PDF-экспорт.
