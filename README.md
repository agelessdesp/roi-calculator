# ROI Calculator for imot.io

Онлайн-калькулятор для быстрой оценки экономического эффекта от внедрения речевой аналитики imot.io.

Открыть калькулятор:

`https://agelessdesp.github.io/roi-calculator/`

## Что умеет калькулятор

- считает ROI и срок окупаемости внедрения;
- сравнивает консервативный, базовый и оптимистичный сценарии;
- учитывает выручку, конверсию, средний чек и дополнительные точки роста;
- помогает подобрать подходящий тариф;
- показывает результат в удобном визуальном виде;
- позволяет экспортировать расчёт в PDF.

## Для кого

Калькулятор подойдёт командам, которые хотят быстро оценить потенциальный бизнес-эффект от внедрения речевой аналитики:

- собственникам;
- коммерческим руководителям;
- руководителям продаж и контакт-центров;
- маркетинговым командам.

## Как использовать

1. Перейдите по ссылке на калькулятор.
2. Заполните основные параметры бизнеса.
3. При необходимости укажите дополнительные вводные по продажам, допродажам и обучению.
4. Выберите тариф или воспользуйтесь рекомендацией калькулятора.
5. Сравните сценарии и сохраните результат в PDF.

## Что лежит в репозитории

- `index.html` — публичная версия калькулятора для GitHub Pages.
- `ROI-калькулятор imotio (версия от 14.04).html` — исходный HTML-файл проекта.
- `AGENTS.md` — внутренняя документация по устройству проекта.

## Технически

Проект сделан как статический одностраничный калькулятор без сборки:

- HTML + inline CSS + inline JavaScript;
- `Chart.js` для графиков;
- `html2canvas` и `jsPDF` для экспорта в PDF.

## Статус публикации

Сайт публикуется через GitHub Pages из ветки `main`.
